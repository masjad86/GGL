Button
<ggl-button> is wrapper and with additional styling. You can enable the button with icon after adding few more properties like showIcon, icon.
<ggl-button label="Success" [showIcon]="showIcon" [showLabel]="showIcon" title="Home" icon="check_circle" type="success" [disabled]="isDisabled" (clicked)="handleButtonClicked($event)"></ggl-button>
You have clicked {{buttonLabel}} button.
Checkbox
This checkbox <ggl-checkbox> is wrapper and with additional styling.
<ggl-checkbox label="Disabled" [checked]="isChecked" [disabled]="isDisabled" (changed)="handleCheckboxChange($event)"></ggl-checkbox>
Checked value: {{isChecked}}
Input
This is <ggl-input>. Input component supports the search feature as well to make the field to search field by setting up few props like enableSearch and provide the search callback.
<ggl-input [readonly]="true/false" [disabled]="true/false" [value]="10000" (valueChange)="textChange()" (valueBlur)="textBlur()" isRequired></ggl-input>
You have entered: {{inputValue}}
Icon
This is <ggl-icon>. IconSize enum will be used to define the size. IconSize enum having option like XXSMALL, XSMALL, SMALL, MEDIUM, LARGE, XLARGE and DEFAULT. The default icon size choosen as IconSize.DEFAULT.
<ggl-icon icon="home" size="18 | 24 | 30 | 36 | 48 | 60 | 72"> </ggl-icon>
Label
</ggl-label> component is the wrapper to display text. You can make the clickable label to enable additional properties like clickable property by setting either true or false and attach (clicked) event
<ggl-label text="Label" [clickable]="isClickable" (clicked)="handleLinkClick($event)" [disabled]="isDisabled"></ggl-label>
You have clicked {{labelClicked}} link.
